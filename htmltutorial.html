HTML lesson 10

Our website is coming along very nicely but I think we can still spice it up a bit. I've added an img tag or "image" <img>. Unlike the other tags the img does not get a closing tag. A src attribute defines the url of the image location. An alt attribute should be added for people with screen readers for photo information. Click on the code to see!

<!DOCTYPE html> <html> <head> </head> <body> <img src="bad_kitty.jpg" alt="bad cat"> </body> </html>
