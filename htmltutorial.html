HTML lesson 1

HTML is an acronym for "Hyper Text Markup Language" and consists of tags like this <html>.
There are a variety of HTML tags and they each define thier own content.
Below is an example of a very simple website.

<!DOCTYPE html> <html> <head> </head> <body> Hello World </body> </html>
