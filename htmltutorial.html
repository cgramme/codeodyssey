HTML lesson 1

HTML is an acronym for "HyperText Markup Language" and consists of tags like <html>.
There are a variety of HTML tags and they each define thier own content.
Below is an example of a very simple HTML website.

<!DOCTYPE html> <html> <head> </head> <body> Hello World </body> </html>
